# Glycan-related Resource Description Framework (RDF) data



# GLIC SPARQL endpoint

GLIC endpoint is a SPARQL endpoint for Glycan-related RDF data.
* GLIC endpoint URL: https://sparql.glycoinfo.org/sparql
